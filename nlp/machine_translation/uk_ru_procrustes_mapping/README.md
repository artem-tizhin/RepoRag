# UK → RU Word Embedding Translator

Цель работы заключалась не в достижении высокого качества перевода,  
а в **практическом понимании принципа отображения эмбеддингов между языковыми пространствами**  
и реализации простого словарного переводчика на основе линейного и ортогонального преобразования.

---

## Описание

В проекте реализован перенос семантических представлений (word embeddings)  
из украинского пространства в русское с помощью **линейного отображения** и  
его улучшенной версии — **ортогонального преобразования Прокруста**.

Использованные этапы:

1. **Загрузка предобученных эмбеддингов** (`cc.uk.300.vec`, `cc.ru.300.vec` из fastText).  
2. **Формирование обучающей выборки** пар слов `(украинское, русское)`.  
3. **Обучение линейного отображения** с помощью `LinearRegression`.  
4. **Реализация ортогонального преобразования** через SVD (задача Прокруста).  
5. **Оценка качества** с метриками `precision@1` и `precision@5`.  
6. **Построение простого переводчика**:  
   каждое украинское слово отображается в общее пространство и заменяется ближайшим русским соседом.

---

## Пример перевода
src: кіт зловив мишу
dst: кот поймал мышку
